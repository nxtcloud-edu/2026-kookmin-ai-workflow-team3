# 이벤트 목록

> "내가 홍명보라면" 주요 이벤트 흐름

## 이벤트 구조

각 이벤트는 **배경 설명 → 선택지 → 결과 이벤트** 순서로 구성된다.
각 이벤트 선택에 따라, 선수단 분위기·국민 민심의 **내부 point**가 변한다.
최종 엔딩은 **이벤트 7**(7-1 ~ 7-8)에서 처리한다.

### 수치 표시 규칙

| 시점 | 플레이어에게 보여줄 것 |
|------|------------------------|
| 선택·경기 진행 중 | **숫자 비공개**. 내러티브 피드백만 표시 (예: "기자들 표정이 굳었다", "라커룸이 조용해졌다") |
| **엔딩** | 최종 **국민 민심·선수단 분위기 수치** + **티어 라벨** + 성적 + 엔딩 제목 |

아래 문서의 `국민 민심`, `선수단 분위기` point는 **설계·구현용 내부 수치**이다. 게임 중 UI에는 노출하지 않는다.

### 수치 티어 (엔딩 표기용)

| 티어 | point 범위 |
|------|------------|
| 붕괴 | ≤ -80 |
| 불안 | -79 ~ -30 |
| 중립 | -29 ~ +29 |
| 상승 | +30 ~ +79 |
| 고조 | ≥ +80 |

- **선수단 분위기** → 경기 결과 판정에 주로 반영
- **국민 민심** → 엔딩 톤·에필로그(여론·기자·협회 반응)에 주로 반영

## 이벤트 5: 2026 북중미 월드컵 조별리그 2차전(VS 멕시코)

- **배경**: 체코와 멕시코가 나란히 1차전에서 승리한 가운데, 이 경기의 승자는 32강 진출을 조기에 확정한다. 멕시코는 홈 관중의 응원을 받지만 주전 중앙수비수 세사르 몬테스가 징계로 결장한다. 홍명보 감독은 기존 전술을 유지할지, 상대 수비의 공백을 적극적으로 공략할지 결정해야 한다.
- **선택지**:
  - A. 체코전과 같은 3-4-3을 유지하고, 황인범을 중심으로 압박을 풀어내며 손흥민·이강인의 전환 공격을 노린다.
  - B. 오현규를 최전방에 선발로 투입하고, 몬테스가 빠진 멕시코 중앙 수비를 롱패스와 빠른 역습으로 집중 공략한다.
- **결과**:
  - A → **실제 결과**: 대한민국은 전반을 0–0으로 마치며 대등하게 맞섰지만, 후반 5분 김승규와 이기혁의 충돌로 흐른 공을 루이스 로모에게 내줘 선제골을 허용한다. 홍명보 감독은 손흥민과 이재성을 빼고 황희찬과 오현규를 투입한 뒤 공격수를 연이어 늘린다. 경기 막판 조규성과 양현준이 결정적인 기회를 만들지만 라울 랑헬의 선방에 막히고, 대한민국은 점유율과 기대 득점에서 앞서고도 0–1로 패한다.
  - B → **가상 결과**: 대한민국은 오현규의 제공권과 버티기를 활용해 멕시코의 대체 중앙 수비진을 계속 압박한다. 오현규가 받아낸 공을 손흥민과 이강인이 빠르게 이어 가며 선제골을 만들고, 후반에는 수비 간격을 좁혀 멕시코의 공세를 막아낸다. 대한민국은 1–0으로 승리해 월드컵 본선에서 처음으로 멕시코를 꺾고, 2연승으로 32강 진출을 조기에 확정한다.

### 이벤트 7-1: 제시마치 (E0)

- **트리거**: 이벤트 1-B (감독직 거절)
- **배경**: 빵집에서 돌아선 뒤, 대한민국 대표팀은 제시마치 감독 체제로 월드컵을 치른다.
- **결과**:
  - **[엔딩 E0] 제시마치**
  - 성적: 32강 진출 (패러디)
  - (연출) *"…그래도 16강— 아니, 32강은 갔다."*

### 이벤트 7-2: 전설의 귀환 (E1-L)

- **트리거**: 16강 이상 + 국민 민심·선수단 분위기 모두 상승 이상 (≥ +30)
- **배경**: 조별리그를 넘어 토너먼트에서도 팀이 들썩인다. 여론은 홍명보 감독을 다시 부른 선택을 후회하지 않는다고 말한다.
- **결과**:
  - **[엔딩 E1-L] 전설의 귀환**
  - 성적: 16강 이상
  - (연출) 에필로그 — *"2014년의 그늘을 완전히 벗어났다"*

### 이벤트 7-3: 명예 회복 (E2-H)

- **트리거**: 32강 진출 + 국민 민심 ≥ +30 + 선수단 분위기 ≥ +30
- **배경**: 조별리그를 통과한 뒤, 기자회견장 분위기가 달라졌다. "말보다 결과"라던 약속이 드디어 믿음으로 바뀌었다.
- **결과**:

  - 성적: 32강 진출
  - (연출) 에필로그 — *"2014의 수모를 씻고 환호받는 감독"*

### 이벤트 7-4: 운 좋은 32강 (E2-I)

- **트리거**: 32강 진출 + 7-3·7-5 조건 미충족
- **배경**: 32강 티켓은 챙겼지만, 논란과 비판은 여전하다. "일단 갔으니 됐다"는 말과 "근본은 안 바뀌었다"는 말이 동시에 올라온다.
- **결과**:

- **배경**: 결과만 보면 통과. 하지만 인터넷과 기자회견은 감독이 아니라 선수들 덕분이라고 말한다. 체코전 B 루트와 잘 맞는 결말이다.
