# 이벤트 목록

> "내가 홍명보라면" 주요 이벤트 흐름

## 이벤트 구조

각 이벤트는 **배경 설명 → 선택지 → 결과 이벤트** 순서로 구성된다.
각 이벤트 선택에 따라, 선수단 분위기/국민 민심의 point 변화가 발생한다.

## 이벤트 4: 2026 북중미 월드컵 조별리그 1차전(VS 체코)

### 이벤트 4-1: 손흥민 교체 (경기 중)

- **배경**: 체코전 후반 20분, 0-1 뒤진 상황. 손흥민은 슈팅은 많았지만 골이 없고, 오현규 교체 카드가 올라왔다. 경기 전 취재진 갈등 여파로 주장과 감독 사이가 미묘하게 어색한 상태다.
- **선택지**:
  - A. "손흥민, 수고했다. 체코가 네 뒷공간만 조이고 있어. 오현규 넣고 박스 높이 올리자."
  - B. "손흥민, 오늘은 골이 없었잖아. 전술 얘기는 집어치우고—네 표정 보니 나도 더는 참을 필요 없겠다. 나와."
- **결과**:
  - A → 2-1 역전승. 선수단 사기 상승, 국민 민심 상승. 이벤트 5(멕시코전) 진행
  - B → 2-1 역전승(분위기 악화). 선수단 사기 저하, 국민 민심 저하, 감독·주장 갈등 발생. 이벤트 5(멕시코전) 진행
