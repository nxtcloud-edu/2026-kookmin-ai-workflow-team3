# 이벤트 목록

> "내가 홍명보라면" 주요 이벤트 흐름

## 이벤트 구조

각 이벤트는 **배경 설명 → 선택지 → 결과 이벤트** 순서로 구성된다.
각 이벤트 선택에 따라, 선수단 분위기·국민 민심의 **내부 point**가 변한다.
최종 엔딩은 **이벤트 7**(7-1 ~ 7-8)에서 처리한다.

### 수치 표시 규칙

| 시점 | 플레이어에게 보여줄 것 |
|------|------------------------|
| 선택·경기 진행 중 | **숫자 비공개**. 내러티브 피드백만 표시 (예: "기자들 표정이 굳었다", "라커룸이 조용해졌다") |
| **엔딩** | 최종 **국민 민심·선수단 분위기 수치** + **티어 라벨** + 성적 + 엔딩 제목 |

아래 문서의 `국민 민심`, `선수단 분위기` point는 **설계·구현용 내부 수치**이다. 게임 중 UI에는 노출하지 않는다.

### 수치 티어 (엔딩 표기용)

| 티어 | point 범위 |
|------|------------|
| 붕괴 | ≤ -80 |
| 불안 | -79 ~ -30 |
| 중립 | -29 ~ +29 |
| 상승 | +30 ~ +79 |
| 고조 | ≥ +80 |

- **선수단 분위기** → 경기 결과 판정에 주로 반영
- **국민 민심** → 엔딩 톤·에필로그(여론·기자·협회 반응)에 주로 반영
