# 홍명보 감독 선택지 대사 생성

플레이어가 연기하는 **홍명보 감독**의 선택지 대사를 쓴다. NPC 대사가 아니다.

## 필수 참고 파일 (먼저 읽기)

1. [Docs/홍명보_캐릭터명세.md](../../Docs/홍명보_캐릭터명세.md) — 성격·말투·비판 맥락
2. [Docs/events.md](../../Docs/events.md) — 기존 이벤트 톤·형식
3. 해당 경기/이벤트 조사 문서 (`Docs/2026_*.md` 등) — 사실 맥락

## 출력 형식 (events.md 준수)

```markdown
- **배경**: [상황 1~2문장. 점수·시간·갈등 요소 포함]
- **선택지**:
  - A. "[감독이 직접 말하는 대사]"
  - B. "[대비되는 의도의 대사]"
- **결과**:
  - A → [경기·사기·민심 등 결과 한 줄]
  - B → [대비 결과 한 줄]
```

- 선택지는 **따옴표 안 감독 1인칭 대사**만 (해설·괄호 설명 최소화)
- 이벤트 1·4-1처럼 **짧고 구체적**으로. 한 선택지 1~3문장
- A/B는 **같은 상황에서 의도만 대비**되게 (둘 다 그럴듯해야 선택이 의미 있음)

## 대사 유형 (선택지 의도별)

| 유형 | 화법 특징 | 게임 효과 예시 |
|------|-----------|----------------|
| **전술·존중** | 이유 설명, 선수 이름 호명, 다음 수 제시 | 사기↑, 민심↑ |
| **두루뭉술·회피** | "준비한 대로", "지켜보겠다", "선수들을 믿는다" | 중립~민심 소폭↓ |
| **책임 전가** | 특정 선수 실수 콕 집음 | 민심↓↓, 기자·팬 반발 |
| **감정·대립** | 짧고 냉정, 전술 설명 생략, 기싸움 | 사기↓, 갈등 |
| **의리·고집** | 익숙한 선수 고수, 변화 거부 | 결과에 따라 극단 분기 |

명세 §2~§3의 패턴을 **그대로 인용하지 말고** 상황에 맞게 변형한다.

## 말투 체크리스트

- [ ] 존댓말·절제된 어조 (감정 폭발은 B·갈등 루트에만 제한적)
- [ ] 실제 감독이 벤치/기자회견에서 말할 법한 문장
- [ ] 풍자 포인트: 능력 제로가 아니라 **결과에 따라 평가가 갈리는 모순** 유지
- [ ] 가상 캐릭터 전제 — 실존 인물 비하가 아닌 게임 시나리오 톤

## A/B 설계 원칙

1. **A = 상대적으로 책임지거나 전술적으로 설명하는 선택**
2. **B = 회피·전가·감정·고집 중 하나를 과하게 드러내는 선택**
3. 결과 문장에 `선수단 사기`, `국민 민심`, `갈등` 등 수치 방향을 명시
4. 실제 역사와 다를 수 있음 — 조사 문서는 **배경**만, 분기는 게임용
